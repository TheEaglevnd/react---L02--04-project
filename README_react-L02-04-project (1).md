# React Practice Project

A React + Vite project created as part of the **Daneshkar Frontend Bootcamp** to practice component structure, state handling, and styling with SCSS.

## About
This repository is a React application scaffolded with Vite, built to reinforce core React concepts such as component composition, state management, and SCSS-based styling within a modern build setup.

## Tech Stack
- JavaScript / React
- SCSS
- HTML / CSS
- Vite + ESLint


## Getting Started
```bash
git clone https://github.com/TheEaglevnd/react---L02--04-project.git
cd react---L02--04-project
npm install
npm run dev
```
Then open the local address shown in your terminal (usually `http://localhost:5173`).

## Build for Production
```bash
npm run build
```
